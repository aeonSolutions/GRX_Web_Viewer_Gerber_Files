# GRX FrontEnd

ODB++ database viewer with 3D showcase

## For Starters

Install dependencies

```
npm install
```

Build react

```
npm start build
```

Start the App

```
npm start
```

Customize the application and build for multiplatform systems

```
npm run build_osx
npm run build_win
npm run build_linux
```
